Description

Telerik UI for ASP.NET MVC enables you to store the custom settings of the user and restore them when the user logs back in at some future moment.

The Telerik for ASP.NET MVC TreeList component enables you to save the applied user preferences by using the getOptions method and then restore them by using the setOptions method. These methods allow you to serialize the current state of the component when needed and recover it later on.

This example shows how to use the localStorage of the browser to save and load the settings of the TreeList.
