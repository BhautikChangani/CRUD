Description

The Telerik UI for ASP.NET MVC MultiSelect supports data virtualization. Virtualization enables the component to display a fixed set of items defined by the user and render the rest of the content within the same items size frame when the list is scrolled, and is useful for displaying large sets of data.

As shown in the demo, the MultiSelect component renders a fixed number of items at once and when scrolled, takes the rest of the items and displays them within the same format. To enable virtualization, create Read and ValueMapper actions, and, then, set the desired number of items that will be rendered.
