Description

The Telerik UI for ASP.NET MVC TreeView supports a comprehensive set of events which enable you to provide fluent experience and smooth interaction with the component. The component exposes various events that allow you to respond to user actions. You can handle any of the events below and execute a custom logic in the respective event handler.

  • Change
  • Check
  • Select
  • Collapse
  • Expand
  • Navigate
  • LoadCompleted
  • Drag-and-drop events—DragStart, Drag, DragEnd, and Drop

The current demo shows how to handle most of the TreeView events and log them in the console log box.
