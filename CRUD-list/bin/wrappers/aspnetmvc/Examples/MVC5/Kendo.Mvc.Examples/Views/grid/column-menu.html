Description

Out of the box, the Telerik UI for ASP.NET MVC Data Grid provides a column menu functionality. The column menu feature allows you to configure a menu with additional options from which the user can pick and also show, hide, filter, and sort the Data Grid columns if filtering and sorting are enabled.

To display the column menu of the ASP.NET MVC Data Grid, use the ColumnMenu(true) method. As a result, the component will render the column menu in its header cells as an ellipsis (a set of three dots), as demonstrated in this demo.

The Telerik UI for ASP.NET MVC Data Grid supports three types of Column Menu - classic, modern, and tabbed. Use the DropDownList to switch between types and examine the different renering.

For more information about the column menu feature of the component, refer to the official Telerik UI for ASP.NET MVC Data Grid product documentation on column settings.
