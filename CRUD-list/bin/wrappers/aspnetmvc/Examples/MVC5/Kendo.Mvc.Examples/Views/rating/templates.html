Description

Telerik UI for ASP.NET MVC provides a user-friendly and high-performing JavaScript templating engine based on the Kendo UI Templates, and also:

  • Represent a substitute for traditional HTML string-building in JavaScript and help you create HTML chunks that can be automatically merged with JavaScript logic.
  • Focus on providing essential templating features for common user interface (UI) rendering scenarios with a heavy emphasis on performance over feature glut by trading convenient syntax sugar for improved performance.

By using the Kendo UI Templates, the Telerik UI for ASP.NET MVC Rating provides full control over the rendering of its items, as well as their selection and hover states.

The current example demonstrates how to configure the ItemTemplate, SelectedTemplate, and HoveredTemplate. All of the template settings are defined through an external template script while using the conventional Kendo UI Template syntax.
