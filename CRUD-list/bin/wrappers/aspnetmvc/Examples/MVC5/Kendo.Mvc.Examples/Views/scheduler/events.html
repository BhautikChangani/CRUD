Description

The Telerik UI for ASP.NET MVC Scheduler supports a comprehensive set of events which enable you to provide fluent experience and smooth interaction with the component.

This demo shows how to handle the following events:

  • DataBound—Fires when the component is bound to data from its data source.
  • DataBinding—Fires before the component binds to its data source.
  • Add—Fires when a new event is about to be added.
  • Edit—Fires when the user opens a Scheduler event in the edit mode or creates a new event.
  • Cancel—Fires when the user cancels editing by clicking the Cancel button.
  • Change—Fires when the user selects a cell or event in the Scheduler.
  • Remove—Fires when the user performs the destroy action.
  • Save—Fires when the user saves a Scheduler event by clicking the Save button.
  • Navigate—Fires when the user changes the selected date or the selected Scheduler view, or switches between rendering the full day and the business hours.
  • MoveStart—Fires when the user starts dragging an event.
  • Move—Fires when the user is moving an event.
  • MoveEnd—Fires when the user stops moving an event.
  • ResizeStart—Fires when the user starts resizing an event.
  • Resize—Fires when the user is resizing an event.
  • ResizeEnd—Fires when the user releases the mouse after resizing an event.
