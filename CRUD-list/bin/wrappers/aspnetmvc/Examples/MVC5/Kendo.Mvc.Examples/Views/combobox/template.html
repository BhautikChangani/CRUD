Description

The Telerik UI for ASP.NET MVC ComboBox is integrated with the user-friendly and high-performing JavaScript templating engine based on the Kendo UI Templates, which also:

  • Represent a substitute for traditional HTML string-building in JavaScript and help you create HTML chunks that can be automatically merged with JavaScript data.
  • Focus on providing essential templating features for common user-interface (UI) rendering scenarios with a heavy emphasis on performance over feature glut by trading convenient syntax sugar for improved performance.
The Telerik UI for ASP.NET MVC ComboBox provides configuration methods to control the rendering of its items, popup header, and footer. As shown in this demo, you can personalize the look and feel of the ComboBox, and include images and other type of customized content with the Header and Item templates.
