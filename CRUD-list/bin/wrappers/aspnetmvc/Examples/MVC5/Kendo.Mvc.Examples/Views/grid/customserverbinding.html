Description

The Telerik UI for ASP.NET MVC Data Grid enables you to bypass the built-in data processing and handle operations such paging, sorting, filtering, and grouping yourself, and provides options for setting custom server-binding and custom Ajax-binding.

This demo shows how to configure a custom server binding in the Data Grid.

For more information about the data-binding capabilities, refer to the official Telerik UI for ASP.NET MVC Data Grid product documentation on the available data-binding approaches.
