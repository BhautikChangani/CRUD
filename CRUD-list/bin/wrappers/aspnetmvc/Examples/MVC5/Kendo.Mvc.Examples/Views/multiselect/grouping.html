Description

The Telerik UI for ASP.NET MVC MultiSelect component supports a grouping feature which helps you organize predefined lists of values into logical groups.

To enable grouping, define a datasource expression with a custom DataSource configuration and specify the field by which the MultiSelect will be grouped. In the current demo, the data of the MultiSelect component is grouped by Country (.Group(g => g.Add("Country", typeof(string)))) and all corresponding customers for that country are listed under each group.
