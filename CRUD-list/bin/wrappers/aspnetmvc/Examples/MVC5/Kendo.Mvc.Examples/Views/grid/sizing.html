Description

The Telerik UI for ASP.NET MVC Grid enables you to reduce its size to achieve better visibility on smaller devices or to render as much data as possible. For a compact appearance, you can change the value of the Size configuration to Small. The compact Grid will utilize the available space by setting smaller padding in its cells.

The Size setting also affects the inner components such as the Toolbar, Pager and editors(TextBox, NumericTextBox and more). Their elements will have smaller padding as well to follow the overall styling of the Grid.
