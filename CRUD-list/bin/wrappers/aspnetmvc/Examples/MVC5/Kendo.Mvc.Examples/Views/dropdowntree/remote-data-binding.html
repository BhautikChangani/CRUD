Description

The Telerik UI for ASP.NET MVC DropDownTree supports remote data binding and you can display remote-storage data in the component.

To implement the remote data-binding feature, you need to specify a remote endpoint or web service returning data in a JSON or JSONP format, and utilize the Telerik UI for ASP.NET MVC DataSource as a mediator between the DropDownTree and the underlying data.

To make the connection to the remote endpoint and process the data properly, the data source requires information about the web service URL.

This example shows how to configure the DropDownTree for remote data binding by adding the DataSource component in the helper definition. By default, the DropDownTree sends to the remote endpoint the id of the expanded node as a query string parameter. In this way, you can filter the data collection in the Action method based on the received id parameter, and return the appropriate child nodes to the DataSource of the component.
