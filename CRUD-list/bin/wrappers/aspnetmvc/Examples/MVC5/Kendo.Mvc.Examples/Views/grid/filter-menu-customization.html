Description

The Telerik UI for ASP.NET MVC Grid enables you to modify its Filter Menu.

This demo shows how to apply common settings to the filter menu configuration of the Data Grid and customize its user interface. To achieve the desired scenario:

  1. Specify a single filter criterion by using the Filterable=>Extra(false) setting and limit the filter operators for string columns to Starts with, Is equal to, and Is not equal to.
  2. After that, provide the custom filterable UI by referencing the name of the JavaScript function which initializes it. The demo uses a Kendo UI AutoComplete and a DropDownList for the Title and City columns respectively—.Filterable(filterable => filterable.UI("cityFilter")).

For more information about the filtering capabilities, refer to the official Telerik UI for ASP.NET MVC Data Grid product documentation on the available fltering approaches.
