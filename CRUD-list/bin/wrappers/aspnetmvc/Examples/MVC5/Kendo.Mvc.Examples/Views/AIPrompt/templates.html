Description

Telerik UI for ASP.NET MVC provide a user-friendly and a high-performing JavaScript templating engine based on the Kendo UI Templates, and also:

  • Represent a substitute for traditional HTML string-building in JavaScript and help you create HTML chunks that can be automatically merged with JavaScript logic.
  • Focus on providing essential templating features for common user-interface (UI) rendering scenarios with a heavy emphasis on performance over feature glut by trading convenient syntax sugar for improved performance.

By using the Kendo UI Templates and the Telerik UI for ASP.NET MVC Template component, the Telerik UI for ASP.NET MVC AIPrompt provides full control over the rendering of its items. In this example, you will find out how to customize the prompt suggestion item and view appearance.
