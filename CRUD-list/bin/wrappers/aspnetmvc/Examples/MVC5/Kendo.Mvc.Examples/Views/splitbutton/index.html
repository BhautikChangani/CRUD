Description

The Telerik UI for ASP.NET MVC SplitButton displays a Button that is bound to an action. It also provides a drop-down list with clickable items.

Features

  • Items—The available options for the items in the SplitButton.
  • Icons—Check how to set icons for the items in the SplitButton.
  • Appearance—The SplitButton supports various styling configurations.
  • Keyboard Navigation—The SplitButton supports a number of keyboard shortcuts for processing various commands.
