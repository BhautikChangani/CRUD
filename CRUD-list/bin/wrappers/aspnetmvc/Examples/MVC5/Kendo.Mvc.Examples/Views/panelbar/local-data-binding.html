Description

The Telerik UI for ASP.NET MVC PanelBar supports local data binding and you can display local-storage data in the component.

The current demo illustrates two approaches for binding the PanelBar to local data:

  • (Default) Using inline data—Pass an IEnumerable of PanelBarItemModel to the BindTo() method. In this example, the list of items is stored in a ViewBag and then passed to the BindTo() option.
  • Mapping Model properties—Bind the PanelBar to a list of objects and map the Model properties to the PanelBarItem properties. This example uses another overload of the BindTo() method as the first argument is an IEnumerable of Model CategoryItem, and the second is the action, which will map the properties.
