Description

The Telerik UI for ASP.NET MVC Grid enables users to work with subsets of Data Grid data such as single ranges, multiple ranges, or specific cells by providing the following capabilities, as demonstrated in this example:

  • Copying and pasting of selected ranges and cells.
  • Exporting data ranges and cells to an MS Excel spreadsheet.
  • Exporting valid data ranges to Telerik UI for ASP.NET MVC Chart

Utilizing the Selection Modes

  • Cell combinations—Using the Ctrl key + left-click combination, selects cells by clicking on them.
  • Range—Left-click on one of the cells and drag the mouse across other cells.
  • Range combination—Make a range selection, press the Ctrl key, and make another range selection.

Using This Demo

  1. Select the cells you want to copy or export.
  2. Click on the menu icon at the top right of the Data Grid.
  3. Select one of the options.
  4. When exporting to the Chart, make sure that both ShipCountry and Freight cells are selected.

For more information about exporting selected content, refer to the official Telerik UI for ASP.NET MVC Data Grid product documentation on export and selection.
