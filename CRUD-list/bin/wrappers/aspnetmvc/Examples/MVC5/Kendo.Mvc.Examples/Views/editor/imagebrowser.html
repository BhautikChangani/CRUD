Descriptions

The Telerik UI for ASP.NET MVC Editor provides the Insert Image and Insert File browser managers to upload and insert Image and Hyperlink objects into the content area of the component. Both dialogs consist of Address and Search fields, a File Browser panel which shows the available files, File Upload, Create a New Folder and Delete features. In addition, there is a file properties section with fields that are suitable for images, such as web address, alternate text, width, and height, as well as for hyperlinks, such as web address and title.

You can configure the Image browser tool through the ImageBrowser configuration option and the Insert File browser through the FileBrowser one. For more information about the Image dialog, refer to the article about the Image Browser.
