Description

The Telerik UI for ASP.NET MVC Menu comes with an extensive set of API configurations that can be used on the client-side to manipulate the behavior of the component. This demo shows how to utilize some of its functions such as enabling and disabling the Menu items, sub-items, and other elements, opening, closing, and removing and adding items, as well as appending items under, before, or after other elements.

You can check the full list of the supported API calls in the ASP.NET MVC Menu API reference documentation.
