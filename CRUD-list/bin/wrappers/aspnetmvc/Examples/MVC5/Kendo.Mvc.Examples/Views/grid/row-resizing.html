Description

The Telerik UI for ASP.NET MVC Data Grid comes with a row-resizing feature, which can be enabled by using the Resizable(r=>r.Rows(true)) method. As a result, users can resize a row by dragging the resize handles of the row. Multiple row resizing is also supported when the multiple rows selection is enabled via the .Selectable(selectable => selectable.Mode(GridSelectionMode.Multiple).Type(GridSelectionType.Row)) method.

For more information about the row-resizing feature of the component, refer to the official Telerik UI for ASP.NET MVC Data Grid product documentation on the row resizing functionality.
