Description

The Telerik UI for ASP.NET MVC DropDownButton displays a Button that is bound to an action. It also provides a drop-down list with clickable items.

Features

  • Items—The available options for the items in the DropDownButton.
  • Icons—Check how to set icons for the items in the DropDownButton.
  • Appearance—The DropDownButton supports various styling configurations.
  • Keyboard Navigation—The DropDownButton supports a number of keyboard shortcuts for processing various commands.
