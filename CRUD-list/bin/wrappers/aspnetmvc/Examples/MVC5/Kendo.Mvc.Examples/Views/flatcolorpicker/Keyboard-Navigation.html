Focus

The Telerik UI for ASP.NET MVC FlatColorPicker is a single-tab-stop component. It implements inner navigation that can be activated by pressing the Enter key. When inner navigation is activated, the focus is trapped within the component. To return to the page navigation, the user must press Esc.

  • Access key + w Focuses the FlatColorPicker.

Keyboard shortcuts available when FlatColorPicker wrapper is focused:

  • Tab navigates to the next focusable element after the FlatColorPicker
  • Shift + Tab navigates to the previous focusable element before the FlatColorPicker
  • Enter Activates the inner FlatColorPicker navigation.

Keyboard shortcuts available when inner FlatColorPicker navigation has been activated:

  • Tab When Gradient view is on, navigates to the next FlatColorPicker element
  • Shift + Tab When Gradient view is on, navigates to the previous FlatColorPicker Element
  • Enter / Space When the Toggle format button is focused, changes the format between RGB and HEX. When a View button is focused, switches to Palette or Gradient view. When a tile in the Palette view is focused it selects that tile color.
  • Arrow Up / Right When any of the FlatColorPicker's sliders is focused, increases its value. When the drag handle in the FlatColorPicker canvas is focused, moves it in the corresponding direction. When Palette view is active, moves focus in the respective direction.
  • Arrow Down / Left When any of the FlatColorPicker's sliders is focused, decreases its value. When the drag handle in the FlatColorPicker canvas is focused, moves it in the corresponding direction. When Palette view is active, moves focus in the respective direction.
  • Esc Returns focus to the wrapper element.
