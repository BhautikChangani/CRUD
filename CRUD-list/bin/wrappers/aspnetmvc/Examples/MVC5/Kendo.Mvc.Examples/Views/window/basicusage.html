Description

The Telerik UI for ASP.NET MVC Window component allows you to render its content in a movable or in a modal window which can be maximized, restored, and closed.

This ASP.NET MVC Window demo shows a basic scenario usage of the component featuring different content types inside such as text and image. The example also showcases the draggable and resizable properties as well as the options for pinning, minimizing and maximizing the Window listed as actions.
