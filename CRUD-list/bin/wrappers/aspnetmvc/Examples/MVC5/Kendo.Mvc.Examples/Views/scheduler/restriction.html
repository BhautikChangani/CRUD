Description

The Telerik UI for ASP.NET MVC Scheduler delivers client-side API options for restricting its events thus providing you with better control over your resources. You can prevent the dragging and resizing of the appointments, and also the creatiing of a new event or the modification of an existing one by criteria defined by you.

For example, in the demo, double-click on the Call Charlie about the project event, change the attendee to Bob and click Save. These actions will result into a This person is not available in this time period. error.

If you change the start date of the HR Lecture event to 10:30 AM, you will also get a This person is not available in this time period. error.
