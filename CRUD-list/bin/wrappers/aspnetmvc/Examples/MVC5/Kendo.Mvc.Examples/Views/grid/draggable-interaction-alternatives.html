Description

The Telerik UI for ASP.NET MVC Data Grid now provides single pointer draggable interactions. This functionality represents an alternative to the built-in draggable interactions thus providing even a greater level of accessibility."

To enable single pointer draggable interactions in the Grid:

  1. Set the ClickMoveClick configuration of the Grid rows—.Reorderable(reorder => reorder.Rows(rows=>rows.ClickMoveClick(true))).
  2. Add a drag handle to a specified column to reorder the rows by dragging—.Draggable(true).

For more information, refer to the official Telerik UI for ASP.NET MVC Data Grid product documentation on dragging and dropping rows and columns.
