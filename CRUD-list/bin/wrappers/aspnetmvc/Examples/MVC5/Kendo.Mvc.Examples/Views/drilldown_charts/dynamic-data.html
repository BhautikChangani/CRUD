Description

The Telerik UI for ASP.NET MVC Chart supports drilldown functionality for exploring data. This demo shows how to create a drilldown chart using inline data.

For more details, see the Drilldown Charts help article.

The ASP.NET MVC Chart component is part of Telerik UI for ASP.NET MVC, a professional-grade UI library with 110+ components for building modern and feature-rich web applications. The UI for ASP.NET MVC Chart is a server-side wrapper for the Kendo UI for jQuery Chart component and supports an HtmlHelper mode.
