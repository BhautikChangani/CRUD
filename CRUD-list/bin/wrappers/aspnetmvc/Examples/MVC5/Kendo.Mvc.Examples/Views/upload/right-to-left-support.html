Description

Right-to-Left (RTL) support represents the ability of a library, website, or application to handle and respond to users who communicate through right-to-left languages. Right-to-left languages are Arabic, Hebrew, Chinese, Japanese, and others.

The default language input for web users can be left-to-right or right-to-left. The Telerik UI for ASP.NET MVC Upload component supports both input modes enabling you to properly render a spectrum of languages.

This demo illustrates how to set the Upload Right-to-Left (RTL) functionality by specifying k-rtl in the div class.
