Description

The Telerik UI for ASP.NET MVC PanelBar allows you to set the following animations effects over its expanded or collapsed items:

  • Enable and disable the animations—Set the Animation() option to true or false.
  • Apply expand animations—Specify the visual animations when the user opens items. You can set a duration through the Expand(expand => expand.Duration()) configuration, and a direction for the effects such as Expand(), Fade(), and so on.
  • Apply collapse animations—Specify the visual animations when the user closes items. You can set a duration through the Collapse(collapse => collapse.Duration()) configuration, and a direction for the effects such as Reverse(), Fade(), and so on.

This demo shows how to apply animations to the PanelBar component, choose between the toggle and expand animation, as well as set the animate opacity if desired.
