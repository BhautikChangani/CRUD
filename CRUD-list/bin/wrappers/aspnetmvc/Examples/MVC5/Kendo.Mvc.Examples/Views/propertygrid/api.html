Description

The Telerik UI for ASP.NET MVC PropertyGrid comes with an extensive set of API configurations. This demo shows how to utilize some of its functions, such as selecting and opening the items in edit mode, expanding and collapsing the groups and sub-groups, and getting the current PropertyGrid model data by using the model() method.

You can check the full list of the supported API calls in the ASP.NET Core PropertyGrid API reference documentation.
