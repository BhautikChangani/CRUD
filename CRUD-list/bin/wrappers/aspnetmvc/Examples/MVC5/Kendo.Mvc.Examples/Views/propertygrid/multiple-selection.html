Description

The Telerik UI for ASP.NET MVC PropertyGrid allows you to display and edit the properties of a specified Model.

This demo shows how to dynamically change the properties of multiple components through the PropertyGrid. Use Ctrl + click to select and deselect the TextBox and Button components. The PropertyGrid will display only the properties of the selected component. When both components are selected, the PropertyGrid will display only their common properties. All changes in the PropertyGrid items will affect the currently selected component.
