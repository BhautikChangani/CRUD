Description

The Telerik UI for ASP.NET MVC ResponsivePanel allows you to customize the way page content is displayed depending on the screen size.

In this demo, you can see the ResponsivePanel in a basic scenario where the sidebar will be hidden on small screens.

The ASP.NET MVC ResponsivePanel control is part of Telerik UI for ASP.NET MVC, a professional-grade UI library with 110+ components for building modern and feature-rich web applications. The ResponsivePanel is a server-side wrapper for the Kendo UI for jQuery ResponsivePanel component and comes in the form of an HtmlHelper.
