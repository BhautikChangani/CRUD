Description

The Telerik UI for ASP.NET MVC DropDownTree comes with a built-in filtering functionality that helps the user to filter the items of the component by their text value. The filtering is disabled by default, and you can enable it by setting the Filter(FilterType.Contains) option. In addition to the Contains filter operator, the DropDownTree also supports StartsWith and EndsWith operators.

This example demonstrates how to implement the filtering functionality of the DropDownTree.
