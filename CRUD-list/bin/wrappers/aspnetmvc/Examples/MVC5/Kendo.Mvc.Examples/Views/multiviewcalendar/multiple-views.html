Description

The Telerik UI for ASP.NET MVC MultiViewCalendar component exposes the ability to control its multiple views for defining the number of horizontal views and months that are displayed at a time.

By default, the MultiViewCalendar renders two views. Using the Views() configuration method enables you to set the desired number of views.

This demo illustrates how to apply multiple views in the MultiViewCalendar through a NumericTextBox control used for user interaction.
