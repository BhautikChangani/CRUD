Description

The Telerik UI for ASP.NET MVC Grid offers the following ways of data binding:

  • Server—The component performs server-side requests (HTTP GET) when doing paging, sorting, and filtering.
  • Ajax—The component will make Ajax requests when doing paging, sorting, filtering, grouping, or saving data.

This demo shows how to implement the server binding of the Data Grid. By default, the Telerik UI Grid for ASP.NET MVC performs server-side requests when doing operations like paging, sorting, and filtering.

For more information, refer to the official Telerik UI for ASP.NET MVC Data Grid product documentation on the available data-binding approaches.
