Description

The Telerik UI for ASP.NET MVC Scheduler exposes rich client-side API and events, which provide easy configuration or extension points for custom functionality on top of the built-in features.

In this demo, you can see how to get a reference to the client-side object of the Scheduler and to use its date and view methods to update its current date and current view.

For more information about the available client-side methods, refer to the kendo.ui.Scheduler API documentation.
