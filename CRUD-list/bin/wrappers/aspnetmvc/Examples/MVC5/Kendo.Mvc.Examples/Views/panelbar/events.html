Description

The Telerik UI for ASP.NET MVC PanelBar supports a comprehensive set of events which enable you to provide fluent experience and smooth interaction with the component. Subscribing to the PanelBar events allows you to react to different user actions, such as clicking on an item, or expanding or collapsing it.

This demo shows how to handle some of the available events for the ASP.NET MVC PanelBar. By interacting with the component and in the console log, you can inspect the Activate, Expand, Collapse, and Select events.
