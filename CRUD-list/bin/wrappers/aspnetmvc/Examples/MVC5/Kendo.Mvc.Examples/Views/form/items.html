Description

The Telerik UI for ASP.NET MVC Form provides the Items configuration that contains the Form fields, their labels and hints, and the editors. You can specify an editor of your choice too, for example, a Telerik UI for ASP.NET MVC AutoComplete, a MaskedTextBox, a DateTimePicker, a CheckBox, or other components. To define the required settings, use the options of the respective editor.

This example demonstrates the basic customization of the Form editors, labels, and hints.
