Description

The Telerik UI for ASP.NET MVC PivotGrid control enables you to export its content to PDF by using the sorting and filtering applied by the user. For greater control over the output, configure the PDF export setting. For instance, you can specify the file name and whether to export all pages, margins, paper size, font, and so on.

You can also customize the appearance of the exported PDF file by using the client-side Kendo UI Drawing library.

The current example illustrates how to configure the PivotGrid for PDF export through the Pdf() configuration method.
