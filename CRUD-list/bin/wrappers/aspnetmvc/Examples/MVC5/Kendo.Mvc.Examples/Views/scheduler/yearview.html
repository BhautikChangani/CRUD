Description

The Telerik UI for ASP.NET MVC Scheduler provides a set of views for defining the layout of its events' rendering. The Year view allows you to display appointments in a twelve-month range.

By clicking on a specific date, a list with all associated events for that date is displayed in a tooltip. Clicking on the tooltip will navigate users to the Day view for the corresponding date.

You can apply further customization for the Year view of the Scheduler by setting its StartDate, Months, and TooltipTemplate properties provided by the SchedulerYearViewBuilder.
