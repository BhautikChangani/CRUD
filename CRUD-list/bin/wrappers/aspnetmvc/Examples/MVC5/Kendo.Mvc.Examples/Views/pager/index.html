Description

The ASP.NET MVC Pager component provides the UI needed to page data and can be easily used alongside other data controls.

The current example displays a common use-case scenario for a Pager component.

The ASP.NET MVC Pager control is part of Telerik UI for ASP.NET MVC, a professional-grade UI library with 110+ components for building modern and feature-rich web applications. The TextBox is a server-side wrapper for the Kendo UI for jQuery Pager component and comes in the form of an HtmlHelper.

Functionality and Features

  • Integration–Example of the Pager integration in a Grid.
  • Keyboard navigation–The Pager provides keyboard support for fast navigation.
