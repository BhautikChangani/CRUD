Description

The Telerik UI for ASP.NET MVC Gantt supports PDF Export out-of-the-box. To provide a simple object model for building and manipulating visual scenes, the PDF export options utilize the Kendo UI for jQuery Drawing graphics library.

In addition, the GanttPdfSettingsBuilder API methods allow you to build on top of the default PDF export options and configure a ProxyUrl, specify the Margins of the exported file, and define whether the links must be avoided.
