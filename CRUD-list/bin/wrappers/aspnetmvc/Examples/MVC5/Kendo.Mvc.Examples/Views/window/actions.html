Description

The Telerik UI for ASP.NET MVC Window allows you to define a non-recognized action name and treats it as a custom action.

This demo shows the implementation approach for capturing and handling click events for custom actions. The ASP.NET MVC Window will generate a custom class for the unfamiliar action based on the name of that action, for example, .Custom("custom"). In this example, the "k-i-custom" class is generated by adding "k-i-" to the name "custom". "k-i-custom" can be used to find the respective custom action.
