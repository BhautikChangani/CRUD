Description

The Telerik UI for ASP.NET MVC Form provides a default and a grid appearance for arranging its layout. When using the default layout, the Form items are organized as a list, while the grid layout option allows you to display the Form items as columns separated by a gutter.

The current demo shows how to configure a Form with a grid layout that contains two columns by setting the Layout option to "grid" and to specify the number of columns and a gutter in-between by using the Cols() and Gutter() methods.
