Description

Telerik UI for ASP.NET MVC Popover provides a user-friendly and a high-performing JavaScript templating engine based on the Kendo UI Templates, and also:

  • Represent a substitute for traditional HTML string-building in JavaScript and help you create HTML chunks that can be automatically merged with JavaScript logic.
  • Focus on providing essential templating features for common user-interface (UI) rendering scenarios with a heavy emphasis on performance over feature glut by trading convenient syntax sugar for improved performance.

By using the Kendo UI Templates, the Telerik UI for ASP.NET MVC Popover provides full control over the rendering of its header and content. In this example, you will find out how to display the desired content by using the Body and Header methods.
