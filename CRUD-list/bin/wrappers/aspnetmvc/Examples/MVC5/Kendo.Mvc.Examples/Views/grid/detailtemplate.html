Description

The Telerik UI for ASP.NET MVC Data Grid provides a detail-template feature that enables you to show additional information for a data item by setting the detail template of the component.

To set up detail template of the Data Grid:

  1. Set the ClientDetailTemplateId which represents the id of the script element which containd the client-side detail template of the Grid view.
  2. Define the script tag content that will be shown when the user expands the Grid rows.

This demo shows how to utilize the detail template feature of the Data Grid by injecting a Telerik UI for ASP.NET MVC TabStrip component with two tabs in the detail template of the Data Grid. The first tab contains the orders of the relevant employee, and the second one holds the employee's contact information.

For more information about the templating capabilities, refer to the official Telerik UI for ASP.NET MVC Data Grid product documentation on the available templates.
