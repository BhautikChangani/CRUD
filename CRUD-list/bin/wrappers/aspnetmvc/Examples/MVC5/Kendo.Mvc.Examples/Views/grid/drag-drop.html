Description

The Telerik UI for ASP.NET MVC Data Grid provides a drag-and-drop feature, which allows users to reorder single or multiple Grid rows by dragging and move rows between different Grids.

To enable the dragging and dropping in the Data Grid:

  1. Set the reorderable configuration of the Grid rows—.Reorderable(reorder => reorder.Rows(true)).
  2. Add a drag handle to a specified column to reorder the rows by dragging—.Draggable(true).

To handle the drag-and-drop operations and modify the data source as required, for example, to move rows between Grids, use the RowReorder event of the Data Grid.

Additionally, you can enable the Navigatable() option and reorder the rows by using the Ctrl + Up Arrow or Down Arrow.

For more information, refer to the official Telerik UI for ASP.NET MVC Data Grid product documentation on dragging and dropping rows and columns.
