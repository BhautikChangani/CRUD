Description

The Telerik for ASP.NET MVC TreeList component delivers a client-side paging functionality which is used when large portions of data are loaded in the component.

This feature requires you to set the following parameters:

  • Define the number of records that the TreeList will display on each page.
  • Define the total number of records in the dataset.
  • Specify the pageSize on the data source and the field in the dataset that will contain the total number of records.

The current example shows how to configure the TreeList for client-side paging.
