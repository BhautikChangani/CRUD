Description

The Telerik UI for ASP.NET MVC Scheduler supports horizontal and vertical grouping of its events by Date. To control the direction of the grouping, configure the Orientation property in the following way: .Group(group => group.Resources("Rooms").Orientation(SchedulerGroupOrientation.Horizontal)), as demonstrated in this example.

The available options of the SchedulerGroupOrientation setting are Default, Horizontal, and Vertical.
