Description

The Telerik UI for ASP.NET MVC ComboBox offers several ways for data binding and one of them is the custom data source approach.

To implement the remote data-binding feature, you need to specify a remote endpoint or web service returning data in a JSON or JSONP format, and utilize the Telerik UI for ASP.NET MVC DataSource as a mediator between the ComboBox and the underlying data.

This demo shows how to configure the data binding of the ComboBox by using the DataSource property and specifying its type as Custom.
