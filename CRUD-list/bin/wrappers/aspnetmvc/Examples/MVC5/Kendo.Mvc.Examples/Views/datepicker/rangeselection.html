Description

The Telerik UI for ASP.NET MVC DatePicker offers date-range selection out-of-the-box. The date-range selection feature can be implemented by using two separate DatePicker components or by using the built-in functionality of the ASP.NET MVC DatePicker.

When visualizing the date-range selection with two separate ASP.NET MVC DatePicker components, render a start and end date with specified minimum and maximum values. To set the value of the DatePickers, use the client-side API and the min() and max() methods. For the complete details of the implementation, review the source code of the demo.
