Description

The Template component is supported in the Scheduler templating options:

  • EventTemplate() and AllDayEventTemplate()—Customize the appearance of the events and the "all day" events.
  • TemplateComponentName() of the Editable() configuration—Pass the name of the custom editor template that is created through the Template component.

This demo shows the implementation approach for using the Template component to define a custom editor template for the events of the Scheduler component and update the default template that renders the events.
