Description

Keyboard navigation provides all web and app users with a fast keyboard-only navigation capability and is also part of the web accessibility features—it enables users with disabilities to fully control their website or app access through the keyboard.

By default, users can only navigate to links, buttons, and form controls with a keyboard. The navigation order in which interactive items receive keyboard focus has to be logical and intuitive. Generally, keyboard navigation logic needs to follow the visual horizontal and vertical flow of the page, for example, left to right and top to bottom, header first followed by the main and then the page navigation.

The keyboard navigation of the Telerik UI for ASP.NET MVC Splitter is always available as demonstrated in this demo and you can navigate through the Splitter with the keyboard. Below is a list of the available key-combinations.

The ASP.NET MVC Splitter supports the following keyboard shortcuts:

Keyboard Legend

  • Alt + W Focuses the first splitbar.
  • Left Arrow In horizontal Splitters, moves the focused splitbar to the left.
  • Right Arrow In horizontal Splitters, moves the focused splitbar to the right.
  • Up Arrow In vertical Splitters, moves the focused splitbar up.
  • Down Arrow In vertical Splitters, moves the focused splitbar down.
  • Ctrl + Left Arrow Collapses the left pane of the Splitter or expands the right one.
  • Ctrl + Right Arrow Collapses the right pane of the Splitter or expands the left one.
  • Ctrl + Down Arrow Collapses the upper pane of the Splitter or expands the lower one.
  • Ctrl + Up Arrow Collapses the lower pane of the Splitter or expands the upper one.
  • Enter Accepts the current position of the splitbar.
  • Esc Returns the splitbar to its initial position.
