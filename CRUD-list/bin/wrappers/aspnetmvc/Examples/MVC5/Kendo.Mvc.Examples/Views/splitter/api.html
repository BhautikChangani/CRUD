Description

The Telerik UI for ASP.NET MVC Splitter comes with an extensive set of API configurations.

The current demo shows some of the available API functions of the ASP.NET MVC Splitter. To modify its panes and explore how their state and size changes, use the options in the Configurator section.

To review the full list of the supported API methods, check the ASP.NET MVC Splitter API reference documentation.
