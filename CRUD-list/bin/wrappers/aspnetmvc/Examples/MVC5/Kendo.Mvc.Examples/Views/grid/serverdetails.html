Description

The server detail template feature of the Telerik UI for ASP.NET MVC Grid allows you to configure a detail template that is rendered on the server.

This example demonstrates how you can utilize the server detail template feature of the Telerik UI for ASP.NET MVC Data Grid when implementing hierarchical table presentation. The key step here is to configure the DetailTemplate setting which sets the template.

For more information, refer to the official Telerik UI for ASP.NET MVC Data Grid product documentation on the server-detail template feature.
