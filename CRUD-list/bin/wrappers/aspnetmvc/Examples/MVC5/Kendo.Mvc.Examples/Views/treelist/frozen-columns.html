Description

The Telerik for ASP.NET MVC TreeList component provides the opportunity to lock (freeze) columns to a specific position, which is always visible while the user scrolls horizontally.

For the feature to work properly, you need to make sure that:

  • Scrolling is enabled.
  • At least one column is initially locked.
  • The height of the TreeList is defined.
  • Explicit pixel widths are set to all columns.
  • The total width of all locked columns is either the same, or less than the width of the TreeList minus three times the width of the scrollbar.
  • The TreeList is not initialized in a hidden container.

The current example shows a basic TreeList configuration with locked columns as well as features like sorting, column resizing, and reordering while having its column menu enabled.
