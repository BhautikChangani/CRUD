Description

The Telerik UI for ASP.NET MVC Grid component exposes an option for defining a content template for its toolbar.

This demo shows a sample implementation of a toolbar template incorporating a Telerik UI for ASP.NET MVC DropDownList to list the Data Grid records by category. The DropDownList is populated with the list of categories and the filter is applied on its change event by invoking the grid.dataSource.filter() method from the client-side API of the DataSource.

Note that the toolbar template is instantiated by the .ToolBar(toolbar => { toolbar.ClientTemplateId("GridToolbarTemplate");}) method. The method expects the id of the script element which holds the client template as a string parameter.

For more information, refer to the official Telerik UI for ASP.NET MVC Data Grid product documentation on the available toolbar options.
