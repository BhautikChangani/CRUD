Description

The Telerik UI for ASP.NET MVC Chip is a compact element that represents an input, attribute, or action. The component enables user input and verifies that input by converting text into chips.

This demo shows how to integrate the Chip in the sushi menu of a restaurant and apply different styling options and icon configurations to the rendered Chip components.

The ASP.NET MVC Chip control is part of Telerik UI for ASP.NET MVC, a professional-grade UI library with 110+ components for building modern and feature-rich web applications. The UI for ASP.NET MVC Chip is a server-side wrapper for the Kendo UI for jQuery Chip component and comes as an HtmlHelper.

Functionality and Features

  • Appearance—The Chip allows you to set different styling options.
  • Icons—You can specify an icon for the Chip content.
  • Avatars—The Chip also enables allows you to add an image related to its label.
  • Removable Chip—You can also remove a Chip when it isn't needed.
