Description

Some scenarios require you to enable users to search for a given value inside the cells of the Telerik UI for ASP.NET MVC Grid.

The Data Grid allows you to implement this requirement by supporting the Search Panel functionality.

This demo is configured to filter the Data Grid columns with the following operators:

  • Order ID—'equal'
  • Freight—'greater than'
  • ShipName—'contains'
  • ShipCity—'contains'

For more information, refer to the official Telerik UI for ASP.NET MVC Data Grid product documentation on implementing a search-panel functionality.
