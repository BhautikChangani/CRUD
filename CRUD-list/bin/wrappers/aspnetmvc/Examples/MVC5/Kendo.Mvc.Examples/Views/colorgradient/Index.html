Description

The Telerik UI for ASP.NET MVC ColorGradient renders a gradient (a hue and an alpha slider) and inputs to manually enter a desired color.

The ColorGradient control is part of Telerik UI for ASP.NET MVC, a professional-grade UI library with 110+ components for building modern and feature-rich web applications. The Telerik UI for ASP.NET MVC ColorGradient is a server-side wrapper for the Kendo UI for jQuery ColorGradient component and comes as an HtmlHelper.

This example demonstrates a configuration for the ColorGradient in a real-world style scenario.

Functionality and Features

  • Formats—The component supports RGB and HEX formats.
  • Contrast Tool—A tool for checking the color contrast ratio between two colors.
  • Accessibility—The ColorGradient is accessible for screen readers and supports WAI-ARIA attributes.
  • Keyboard Navigation—The ColorGradient supports a number of keyboard shortcuts for processing various commands.
