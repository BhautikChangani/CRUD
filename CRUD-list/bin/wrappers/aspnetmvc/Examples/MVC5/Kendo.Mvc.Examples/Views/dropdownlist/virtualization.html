Description

The virtualization feature is useful for performance optimization when handling a large amount of data. To enable it you need to configure the .Virtual property, create Read and ValueMapper actions and set the desired number of items to be rendered. When using virtualization the Telerik UI for ASP.NET MVC DropDownList displays a fixed number of items from a large data set and when scrolled it loads and renders to the user the next batch of items ensuring smooth and fast performance.

This Telerik UI for ASP.NET MVC demo shows you how to setup and use the virtualization feature of the component.
