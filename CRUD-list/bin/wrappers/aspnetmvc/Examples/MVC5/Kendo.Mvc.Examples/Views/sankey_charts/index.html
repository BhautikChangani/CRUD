Description

The Telerik UI for MVC Sankey Chart allows you to create diagrams that visualize changing flows and their distribution between domains. Sankey diagrams suit a variety of use cases like the representation of website traffic, budget breakdowns, energy flow, and others.

The ASP.NET MVC Sankey component is part of Telerik UI for ASP.NET MVC toolset, a professional-grade UI library with 110+ components for building modern and feature-rich web applications.

Functionality and Features

  • Customization—You can configure the labels, nodes, and links displayed on the Sankey Diagram.
