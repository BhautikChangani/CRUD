Description

The Telerik UI for ASP.NET MVC ComboBox supports the implementation of cascading ComboBoxes. A cascading ComboBox is a series of two or more ComboBoxes in which each ComboBox is filtered based on the selected options in the previous ComboBox.

This demo shows three ComboBoxes, each enabled and filtered according to the user selection in the previous field. The child components are disabled and have a CascadeFrom property with a specified category from which they draw information and filter the data.
