Description

The Telerik UI for ASP.NET MVC Grid provides options for displaying table data in hierarchical-like fashion so that users can visualize relations between parent and child records. This is where the Telerik UI for ASP.NET MVC Grid hierarchy feature comes into play.

This demo shows how to implement the server hierarchy feature of the Data Grid.

For more information, refer to the official Telerik UI for ASP.NET MVC Data Grid product documentation on hierarchy.
