Description

The Telerik UI for ASP.NET MVC PanelBar comes with an extensive set of API methods. After the initialization of the component, you can get its reference and use the API methods to modify the structure and state of the PanelBar items.

You can check the full list of the supported API calls in the ASP.NET Core PanelBar API reference documentation.

In this demo, you can observe the functions in action by choosing between the expand or collapse feature of an item, add or remove items and sub-items, choose where to add the items, whether before or after the selected ones, as well as enable or disable a whole panel section.
