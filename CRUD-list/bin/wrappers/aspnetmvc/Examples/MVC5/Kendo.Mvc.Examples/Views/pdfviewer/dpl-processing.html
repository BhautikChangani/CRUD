Description

The Telerik UI for ASP.NET MVC PDFViewer component enables you to choose the Document Processing Library as a PDF framework, thus providing you with a complete set of APIs for further procesing the PDF file on the server-side.

The current example illustrates how to load a PDF file directly within the PDFViewer by using the Document Processing Library and how to enable the paging capabilities for loading PDF pages on demand.

DPL Processing Limitations

The DPL Processing library has the following limitations:

  • Clipping elements and Gradients are currently not supported.
  • SVG or encoded images in other formats may not render correctly.
