Description

The Telerik UI for ASP.NET MVC Grid component delivers the hierarchy feature, which enables you to display the table data in a hierarchical-like fashion visualizing, in this way, the relations between parent and child records.

To implement hierarchy in the Data Grid:

  1. Set the ClientDetailTemplateId which specifies the id of the script element containing the client-side detail template of the ASP.NET MVC Data Grid.
  2. Define the script tag content that will be shown when the user expands the rows of the Data Grid.

For more information about the hierarchy feature, refer to the official Telerik UI for ASP.NET MVC Data Grid product documentation on the available hierarchy implementations.
