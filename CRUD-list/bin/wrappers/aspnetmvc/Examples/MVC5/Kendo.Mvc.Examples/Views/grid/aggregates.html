Description

Some development scenarios require the Telerik UI for ASP.NET MVC Grid to display results from aggregate calculations at its bottom groups or columns.

The ASP.NET MVC Data Grid comes with built-in support for the most popular aggregate functions such as Average, Count, Sum, Min, Max, and Count. To enable the required aggregate functions, define the aggregates through the available ClientGroupHeaderColumnTemplate, ClientFooterTemplate, and ClientGroupFooterTemplate Columns templates, and the Group and Aggregate fields of its DataSource instance.

For more information on the aggregates feature, refer to the documentation about managing the group templates of the Telerik UI for ASP.NET MVC Data Grid.
