Description

The Telerik UI for ASP.NET MVC Grid provides a server edit mode, as demonstrated in this example, which uses form elements internally when server editing is enabled. This implementation means that you are not able to place the Grid in another form element on the page because the nesting of forms is not a standard-compliant characteristic.

For more information, refer to the official Telerik UI for ASP.NET MVC Data Grid product documentation on the available server-editing approaches.
