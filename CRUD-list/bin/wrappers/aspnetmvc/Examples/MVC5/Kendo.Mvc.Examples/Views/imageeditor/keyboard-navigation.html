  • Alt + w focuses the widget

Actions Applied to the ImageEditor Toolbar

For its toolbar, the ImageEditor integrates the available keyboard shortcuts of the Kendo UI for jQuery ToolBar component.

Supported keys and user actions in the rest of the component

  • TAB focuses next focusable item
  • Shift + TAB focuses previous focusable item
  • Enter presses the focused button
  • Space presses the focused button
  • Ctrl + z undo
  • Ctrl + y redo
