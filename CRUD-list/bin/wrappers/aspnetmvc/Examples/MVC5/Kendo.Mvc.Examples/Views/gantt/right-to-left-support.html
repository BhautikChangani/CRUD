Description

Right-to-Left (RTL) support represents the ability of a library, website, or application to handle and respond to users who communicate through right-to-left languages. Right-to-left languages are Arabic, Hebrew, Chinese, Japanese, and others.

The default language input for web users can be left-to-right or right-to-left. The Telerik UI for ASP.NET MVC Gantt supports both input modes enabling you to properly render a broader spectrum of languages, thus making your app truly global and helping your localization efforts.

This demo shows how to set the Gantt RTL functionality by setting the k-rtl class in the div Html element that contains the Gantt.
