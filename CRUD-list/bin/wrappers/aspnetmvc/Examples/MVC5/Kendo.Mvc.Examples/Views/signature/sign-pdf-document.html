Description

This demo about the Telerik UI for ASP.NET MVC Signature component demonstrates how to save the Signature within a PDF file.

It makes use of the capabilities provided by the Telerik Document Processing Libraries to extract the PDF file within a memory file and then re-apply the Signature bytes to create a new PDF document.

The initial PDF file is conveniently displayed to the user with another component called Telerik UI for ASP.NET MVC PDFViewer. This component also provides a built-in Download Button which can be used to Save the new PDF document with the new Signature applied.
