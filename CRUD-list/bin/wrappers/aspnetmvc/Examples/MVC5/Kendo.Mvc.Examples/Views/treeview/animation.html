Description

The Telerik UI for ASP.NET MVC TreeView allows you to apply animations effects when expanding or collapsing its nodes:

  • Enable and disable the animations—Set the Animation() option to true or false.
  • Apply expand animations—Specify the visual animations when the user opens a node. You can set a duration through the Expand(expand => expand.Duration()) configuration, and a direction for the effects such as Expand() and Fade().
  • Apply collapse animations—Specify the visual animations when the user closes a node. You can set a duration through the Collapse(collapse => collapse.Duration()) configuration, and a direction for the effects such as SlideIn() and Fade().

This demo shows how to apply animations to the TreeView component, choose between the toggle and expand animation, as well as set the animate opacity if desired.
