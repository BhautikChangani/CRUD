Description

Bind Telerik UI for ASP.NET MVC DropDownList to a wide range of data sources: static arrays, local or remote data, or even Web API endpoints. You can configure the data binding in just a few steps.

The Telerik UI for ASP.NET MVC DropDownList supports several approaches for Data Binding including binding to remote data as shown in this demo example - the list of products is bound to data model retrieved from a database.
