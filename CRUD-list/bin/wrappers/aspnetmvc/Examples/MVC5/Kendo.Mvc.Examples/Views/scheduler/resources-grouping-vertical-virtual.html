Description

The Telerik UI for ASP.NET MVC Scheduler provides the vertical virtualization feature, which optimizes the rendering of the view by doing a DOM virtualization. The vertical virtualization functionality is available for the Day, Week, WorkWeek, and Month views.

You can enable vertical virtualization by using the Virtual(true) method in the view configuration in the following way: views.WeekView(view => view.Selected(true).Virtual(true));.

When the feature is enabled, the view will render additional DOM elements as you scroll down.
