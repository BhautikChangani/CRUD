Description

The Telerik UI for ASP.NET MVC Grid supports the implementation of the CRUD (Create, Read, Update, Destroy) data operations and a WebAPI controller. A WebAPI is a concept referring to APIs over the web, which can be accessed through the HTTP protocol and which are typically used with JavaScript. A WebAPI controller manages the incoming HTTP requests and sends back a response to the caller.

This demo shows how to bind the Data Grid in a WebAPI. By specifying the DataSource type as .WebApi(), the requests will be sent to the endpoints of the WebAPI in the expected format out of the box.

For more information, refer to the official Telerik UI for ASP.NET MVC Data Grid product documentation on WebAPI editing.
