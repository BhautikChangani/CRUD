Description

The supports remote data binding and you can display remote-storage data in the component.

To implement the remote data-binding feature, you need to specify a remote endpoint or web service returning data in a JSON or JSONP format, and utilize the Telerik UI for ASP.NET MVC DataSource as a mediator between the ListView and the underlying data.

To make the connection to the remote endpoint and process the data properly, the data source requires information about the web service URL.

In this example, the ListView is bound to remote data and visualizes the data by using a template.
