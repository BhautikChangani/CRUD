Description

The Telerik UI for ASP.NET MVC Data Grid enables you to bypass the built-in data processing and handle operations such paging, sorting, filtering, and grouping yourself.

This demo shows how to retrieve a custom response in the Data Grid when the use of the server-side extensions is not desired. The requests are sent in the expected format by the controller but the response is processed programmatically and the custom result data set is passed to the Telerik UI for ASP.NET MVC Grid component.

For more information about the data-binding capabilities, refer to the official Telerik UI for ASP.NET MVC Data Grid product documentation on the available data-binding approaches.
