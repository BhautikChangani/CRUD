Description

The Telerik UI for ASP.NET MVC PivotGridV2 provides the built-in functionality to export its data in a PDF file through its client-side saveAsPDF method. This feature comes in handy when it is required to extract and download reports, tabular, and presentation data.

Under the hood, to provide a simple object model for building and manipulating visual scenes, the PDF export options utilize the Kendo UI for jQuery Drawing graphics library.
