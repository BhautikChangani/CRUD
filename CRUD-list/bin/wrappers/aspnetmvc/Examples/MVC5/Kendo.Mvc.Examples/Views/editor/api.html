Description

The Telerik UI for ASP.NET MVC Editor comes with an extensive set of API configurations and exposes a rich set of events, which allow you to execute custom functionality on the client-side on top of the built-in component features.

This example shows how to get the client instance of the Editor and obtain or replace its content by using the editor.value() method.

You can check the full list of the supported API calls on its client-side API page.
