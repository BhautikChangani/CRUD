Description

Many use cases require a customized look and feel or to display an option differently based on a specific condition. For example, if you have a PanelBar bound to remote data, you may need to flag some of the items based on the value of a specified Model property. In such scenarios, it comes in handy to use a custom template to modify the appearance of each PanelBar item dynamically.

This demo shows how to render the PanelBar items through an external Kendo UI Template by using an if-else statement to identify the child items and add a Delete link or icon. The logic for deleting child items is implemented with jQuery after the PanelBar initialization.
