Description

The Telerik UI for ASP.NET MVC ToggleButton component enables you to group related options. The component delivers multiple options to display various buttons depending on your specific use case.

The ToggleButton control is part of Telerik UI for ASP.NET MVC, a professional-grade UI library with 110+ components for building modern and feature-rich web applications. The Telerik UI for ASP.NET MVC ToggleButton is a server-side wrapper for the Kendo UI for jQuery ToggleButton component and comes in the form of an HtmlHelper.

This example demonstrates the basic configuration for the ToggleButton.

Functionality and Features

  • Appearance—The ToggleButton can alter its appearance by using a variety of sizing, outline, and themecolor options.
  • Icon—The ToggleButton can accommodate an icon which enhances the visual representation of the text content.
  • Badge—The ToggleButton can accommodate a badge to boost its text content.
