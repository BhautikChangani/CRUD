Description

The Telerik UI for ASP.NET MVC ListView supports local data binding and which enables you to pass an arbirary model directly within the boundaries of the component.

To implement the local data-binding feature, you need to return a collections model and utilize the Telerik UI for ASP.NET MVC DataSource to enable to operations to execute on the client.

In this example, the ListView is bound to local data and visualizes the data by using a template.
