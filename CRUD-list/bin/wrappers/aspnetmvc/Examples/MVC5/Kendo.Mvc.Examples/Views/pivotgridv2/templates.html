Description

The Telerik UI for ASP.NET MVC PivotGridV2 provides the built-in functionality to change the appearance of the column headers, row headers, and cell values.

The PivotGridV2 is integrated with Kendo UI Templates—a high-performing JavaScript templating engine that allows you to:

  • Represent a substitute for traditional HTML string-building in JavaScript and help you create HTML chunks that can be automatically merged with JavaScript logic.
  • Focus on providing essential templating features for common user-interface (UI) rendering scenarios with a heavy emphasis on performance over feature glut by trading convenient syntax sugar for improved performance.

The ColumnHeaderTemplate renders the content of the column header cell. By default, it renders the caption of the tuple member.

The RowHeaderTemplate renders the content of the row header cell. By default, it renders the caption of the tuple member.

The DataCellTemplate is the template which renders the content of the data cell. By default, it renders the fmtValue formatted value of the data item.
