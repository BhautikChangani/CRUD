Description

The control comes with an option for a single or multiple selections of items, disabled by default.

You can enable the single selection with the .Selectable(ListViewSelectionMode.Single) configuration option. The Multiple Selection can be enabled by simply switching the Mode from Single to Multiple.

To make the connection to the remote endpoint and process the data properly, the data source requires information about the web service URL.

In this demo, you can view the output of the performed selection in the console log. To try out the Multiple Selection option, press and hold the Ctrl key while selecting items.
