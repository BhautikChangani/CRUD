Description

The Telerik UI for ASP.NET MVC Grid provides the built-in functionality to conditionally make a column visible, based on the width of the Browser window.

As shown in this demo, you can control the visibility of the columns by using their media property. media accepts valid strings for the matchMedia browser API and toggles the visibility of the Data Grid columns based on the media queries.

For more information about other column options, refer to the official Telerik UI for ASP.NET MVC Data Grid product documentation on the available column-management approaches.
