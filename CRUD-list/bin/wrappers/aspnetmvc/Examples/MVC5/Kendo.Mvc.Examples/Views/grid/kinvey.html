Description

Kinvey Backend Services are a set of cloud-based services that enable developers to build and manage mobile apps significantly faster, easier, and in a more scalable way. Kinvey Backend Services can significantly speed up development by providing your app with data storage, user management, and server-code execution.

The Telerik UI for ASP.NET MVC Grid supports the data-binding approach which enables the component to consume data from the Kinvey Backend Services.

Note that the Kinvey Backend Services data source is not yet supported on IE11 and earlier because it relies on cross-origin resource sharing.

This demo shows how to bind the Telerik UI for ASP.NET MVC Grid to a Kinvey Backend Service.

For more information about the Kinvey Backend Services data-binding options, refer to the folowing resources:

  • Kinvey Backend Services Integration with Kendo UI
