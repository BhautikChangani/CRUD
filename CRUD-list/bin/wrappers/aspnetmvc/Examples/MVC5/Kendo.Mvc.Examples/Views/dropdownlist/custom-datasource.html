Description

Bind the Telerik UI for ASP.NET MVC DropDownList to a wide range of data sources: static arrays, local or remote data, or even Web API endpoints. You can configure the data binding in just a few steps.

From the demo you can see how to configure the Telerik UI for ASP.NET MVC DropDownList for data binding by using a custom DataSource and binding it to a ToDataSourceResult instance.
