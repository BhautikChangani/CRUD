Description

The Telerik UI for ASP.NET MVC DateInput component enables you to enter dates and time via direct input. It provides separate sections for day, month, year, hours and minutes. You can customize the date and time format, the placeholder messages and the UX when typing dates.

Features

  • Customizing appearance
  • Floating Label

You can find additional information on how to use the UI for ASP.NET MVC DateInput in this section of the product documentation.
