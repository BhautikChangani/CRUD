Description

The Telerik UI Upload for ASP.NET MVC component supports file validation based on their size and extensions.

This demo example shows the AllowedExtensions configuration that accepts the listed types of file that are allowed for upload as an array of strings. The last example demonstrates the MaxFileSize() option which does not allow upload of files that exceed the set limit. In addition, there is also a MinFileSize() option which does not allow files that are less than the set minimum size.
