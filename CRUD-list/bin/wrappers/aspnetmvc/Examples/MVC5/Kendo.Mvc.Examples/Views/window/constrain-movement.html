Description

The Telerik UI for ASP.NET MVC Window allows you to constrain its movement inside a specific container element. This functionality is enabled through the .Draggable(settings => settings.Containment() configuration option.

The Containment option has the following specifics:

  • Overrides the AppendTo setting.
  • The Containment element must have a set CSS position such as relative, absolute, or fixed.

This demo shows how to constrain the movement of the Telerik UI for ASP.NET MVC Window inside a container element. To enable the feature, use the Containment() option and specify the position of the container by using relative, absolute, or fixed CSS rules.
