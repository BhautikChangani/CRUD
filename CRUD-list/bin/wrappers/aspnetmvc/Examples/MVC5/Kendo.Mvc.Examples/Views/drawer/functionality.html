Description

The Telerik UI for ASP.NET MVC Drawer supports the following modes that are different in terms of behavior:

  • Overlay—Takes the entire height of the page and provides an overlay effect on the page when the user opens it.
  • Push—The Drawer interacts with specific content or sections on the page. When the page is opened, depending on the Drawer's position, it pushes the content to the left or right to occupy the remaining space. The associated content shrinks.

The Drawer component can be positioned either on the left or right side by using the Position configuration option.

This demo shows how to render the Drawer based on its selected mode and position.
