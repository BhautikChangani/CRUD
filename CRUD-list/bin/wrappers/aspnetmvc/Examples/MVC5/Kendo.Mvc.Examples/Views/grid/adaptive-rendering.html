Description

The Telerik UI for ASP.NET MVC Grid supports the adaptive mobile rendering feature, which you can enable by setting its Mobile attribute to true. As a result, the Grid will display the column and filter menu, and the popup edit form slider in a new screen for the user. You can force the mobile rendering for all browser types by setting the mobile configuration to phone.

The column resizing feature on touch-screen devices is triggered when the user taps and holds the respective column header. When the Resize icon appears, the user can resize the column by dragging.

For more information on the adaptive rendering feature, refer to the documentation about managing the responsive web design of the Telerik UI for ASP.NET MVC Data Grid.
