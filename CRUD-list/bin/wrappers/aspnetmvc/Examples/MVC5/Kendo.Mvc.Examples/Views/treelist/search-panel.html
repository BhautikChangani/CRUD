Description

Out of the box, the Telerik TreeList for ASP.NET MVC component enables the users to search through its data by using the search panel.

Under the hood, the search panel uses filtering to show only the relevant records in the TreeList.

To enable the search panel functionality, include the Search option to the configuration of the ToolBar:

.ToolBar(t => t.Search())

You can also customize which fields to search through the data when a value is entered in the search input:

.Search(s=> { s.Field(c => c.ContactName); })
