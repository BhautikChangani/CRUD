Description

The Telerik UI for ASP.NET MVC DockManager supports a comprehensive set of events which enable you to provide fluent experience and smooth interaction with the component.

This demo shows how to handle the following events:

  • Dock—triggered when a pane is docked.
  • Pin—triggered when a pane is pinned.
  • Unpin—triggered when a pane is unpinned.
  • InnerDock—triggered when a pane is docked inside the current pane.
  • Resize—triggered when a pane is resized.
  • Drag—triggered when a pane is dragged.
  • DragStart—fired when item drag starts.
  • DragEnd—fired when item drag ends.
  • Close—triggered when a pane is closed.
