Description

Enhance the user experience with efficient filtering. The filter kicks in immediately after you start typing. With only a few settings, you can quickly adjust the Telerik UI for ASP.NET MVC DropDownList for client or server filtering.

The Telerik UI for ASP.NET MVC DropDownList offers filtering options – server or client – which eases the searching process and narrows down the values that you’re interested in. This demo showcases the server filtering which is useful for heavy data rendering and is configured by setting the ServerFiltering property to true.
