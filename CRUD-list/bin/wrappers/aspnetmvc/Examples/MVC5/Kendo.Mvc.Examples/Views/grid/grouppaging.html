Description

The Telerik UI for ASP.NET MVC Grid supports a grouping functionality for the data records it displays.Grouping allows you to page through groups and load the group items on demand. The loading of the group items is executed when a group is expanded. Group paging works with both local and remote data.

This demo shows how to implement a remotely-bound Data Grid with paging and enabled GroupPaging configuration.

For more information about the grouping functionality, refer to the official Telerik UI for ASP.NET MVC Data Grid product documentation on the available aggregates and group paging approaches.
