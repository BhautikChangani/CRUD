Description

The Template component can be used to define nested components inside the tiles of the TileLayout. You can display any Telerik UI for ASP.NET MVC component through the AddComponent() method of the Template. Also, use the AddHtml() option to insert HTML code between the components.

This example demonstrates how to utilize the Template component to integrate an inline editable Grid with a custom DropDopwnList editor and Line chart inside the TileLayout.
