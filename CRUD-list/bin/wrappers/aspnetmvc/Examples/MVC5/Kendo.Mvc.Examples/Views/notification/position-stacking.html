Description

The Telerik UI for ASP.NET MVC Notification provides options to customize its position to left, right, top and pinned, through the .Position configuration.

In case multiple notifications need to be displayed, you can define the direction in which they will stack with regard to the first notification. The possible stacking values are Up, Right, Down, Left, and Default. To configure stacking, use the .Stacking option and assign a value from the NotificationStackingSettings enum members.
