Description

The Telerik UI for ASP.NET MVC MultiViewCalendar component comes with a comprehensive set of API configurations.

This example demonstrates how to configure some of its functions such as selecting a specific range of dates or the navigation within the calendar view to future, past, upper view, and lower view states.

You can check the full list of the supported API calls in the ASP.NET MVC MultiViewCalendar API reference documentation. The code is available for the HtmlHelper mode of the component.
