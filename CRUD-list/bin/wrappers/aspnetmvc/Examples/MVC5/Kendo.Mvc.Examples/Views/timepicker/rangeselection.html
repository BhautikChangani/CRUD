Description

The Telerik UI for ASP.NET MVC TimePicker component provides the opportunity to use a range selection.

This demo shows how to implement the range selection functionallity of the TimePicker. By setting By setting the Min and Max configurations set to render a Start and an End TimePicker, you define only the possible selection ranges.

By using the Change event of the Start TimePicker, you can manipulate the Max configuration of the End Timepicker to change the timestamp selection range.
