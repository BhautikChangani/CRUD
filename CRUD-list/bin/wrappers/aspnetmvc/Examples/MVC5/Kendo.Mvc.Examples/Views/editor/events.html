Description

The Telerik UI for ASP.NET MVC Editor supports a comprehensive set of events which enable you to provide fluent experience and smooth interaction with the component, and provide extension points for customization on top of its built-in features.

This demo shows how to intercept the Change, Execute, Select, and Paste events of the Eitor to output messages in the console when these events are raised. You can use these events for customizations or to execute custom logic, if necessary.
