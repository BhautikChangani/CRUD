Description

The Telerik UI for ASP.NET MVC Scheduler allows you to assign Scheduler events to a set of predefined resources. The resources can be divided in the following way:

  • Single-instance resourcesâ€“When only a single instance can be assigned to a Scheduler event. This, for example, can be a list of meetings (Scheduler events), which are held in two rooms (resources). Since a meeting can be held in only one room, it can be considered a single resource.
  • Multiple-instance resources-When two or more instances can be assigned to a Scheduler event. This, for example, can be a list of meetings and meeting attendees. Since more than one attendee can participate in a meeting, it is considered a multiple-instance resource. You can enable the multiple-instance feature by setting the Resources configuration to .Multiple(true).

The Scheduler also supports unlimited resource types. For example, you can combine single and multiple resources, as shown in this demo.
