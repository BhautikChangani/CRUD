Description

The Telerik for ASP.NET MVC TreeList component provides an option to display built-in aggregate calculations when the supplied data is grouped. You can display the aggregated result in a group header, footer, or in the summary footer of the component.

The greatest thing is that with Telerik UI for ASP.NET MVC you can customize the content of the aggregates either by using the external JavaScript mechanism or by conventional hash syntax. Both approaches are based on the Kendo UI Templates.

The current example showcases how to set the aggregates in the TreeList by using the FooterTemplate() configuration method.
