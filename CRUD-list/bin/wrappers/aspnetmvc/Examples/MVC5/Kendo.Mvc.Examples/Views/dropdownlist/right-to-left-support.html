Description

Use the Telerik UI for ASP.NET MVC DropDownList control in any application. Even if you design it for languages like Arabic or Hebrew, where users read from right to left. With only a few settings, you can adjust the RTL (Right to Left) feature and rearrange the DropDownList in your application.

The demo shows the built-in option for right-to-left rendering for the Telerik UI for ASP.NET MVC DropDownList component configured in the div class.
