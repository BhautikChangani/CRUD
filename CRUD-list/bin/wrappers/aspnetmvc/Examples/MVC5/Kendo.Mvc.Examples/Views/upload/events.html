Description

The Telerik UI Upload for ASP.NET MVC component exposes a large set of events such as Complete, Upload ,Error and more. All handled in the Console Log when you interact with the component.

This demo showcases how to handle the Select event that is fired when the user selects a file to be uploaded in the component, the Upload that is triggered when a file is about to be uploaded, and the Progress event fired when the data about the progress of the uploaded file is available.
