Description

The Telerik UI for ASP.NET MVC Loader is a visual indicator that expresses an indeterminate wait time. The component informs the users about the status of ongoing processes, such as loading an application, submitting a form, saving updates, or fetching data. Besides its basic configuration options, the Loader provides accessibilityâ€”it comes with WAI-ARIA compatibility, and screen-reader users can also interact with the component.

This demo shows the different Loader animations and how to change them dynamically when a button is clicked.

The ASP.NET MVC Loader control is part of Telerik UI for ASP.NET MVC, a professional-grade UI library with 110+ components for building modern and feature-rich web applications. The UI for ASP.NET MVC Loader is a server-side wrapper for the Kendo UI for jQuery Loader component and comes in the form of an HtmlHelper.

Functionality and Features

  • Appearance—The component allows you to set different styling options.
  • Integration—You can integrate the Loader into other UI components.
  • Accessibility—The Loader supports various accessibility standards.
