Description

The Telerik UI for ASP.NET MVC ButtonGroup comes with an extensive set of API configurations. This demo shows how to utilize some of its functions, such as enabling and disabling the component through the enable() client-side method, and selecting a specified button by index with the help of the select() method. The ButtonGroup also exposes methods for setting a Badge to the button item and getting the currently selected button.

You can check the full list of the supported API calls in the ASP.NET MVC ButtonGroup API reference documentation.
